# PROJETO — ix.BLZ-S(DS+UTM)

## Visao Geral

Projeto de sincronizacao, enrich e analise de dados comerciais com foco em DealScore, UTM e automacao operacional.

## Stack

- Python
- HTML, CSS, JavaScript
- Templates server-rendered
- Integracoes: Pipedrive, Pushover, Linear

## Estrutura

- main.py
- dealscore/
- scripts/
- templates/
- static/
- modulos/mopgled/
- _docs/

## Features

- DealScore e enrich em lote
- UI operacional de sync e datas
- Integracao Mopgled
- Operacao padronizada com ix.WP

## Setup

1. Configurar .env e .env.wp.
2. Instalar dependencias.
3. Executar python3 main.py.
