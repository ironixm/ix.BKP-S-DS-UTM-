# MopGled Client (módulo)

Este diretório contém o módulo cliente e o instalador principal.

Para instalação guiada, use os wrappers na raiz do pacote:

```bash
python3 instalador/install-py.py --project /caminho/do/projeto
```

Ou rode diretamente:

```bash
python3 modulos/mopgled-client/instalar.py --project /caminho/do/projeto
```
