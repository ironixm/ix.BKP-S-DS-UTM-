# AGENTE – Migrations

## Regras
- Use Alembic via Flask-Migrate.
- Sempre versionar alterações de modelos.
- Nunca editar migrations antigas sem justificativa.

## Comandos
- `flask db init`
- `flask db migrate -m "descricao"`
- `flask db upgrade`
