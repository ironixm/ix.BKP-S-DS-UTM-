# AGENTE – Core

## Responsabilidades
- Inicialização do app Flask.
- Registro de blueprints e healthcheck.
- Configuração de login e filtros Jinja.

## Arquivos
- `factory.py` – cria o app e carrega extensões.
- `routes_loader.py` – registra rotas base.
- `routes_base.py` – rotas públicas básicas.
- `health.py` – endpoint `/health`.
