# ix.BZP-DealScore-UTM-Sync

Projeto inicial criado via MopGled Client.

## Rodar local
```bash
./_Start(▶︎)_.command
```

## MopGled
- Por padrão usa `static/mopgled.css` e `static/mopgled.js`.
- Para usar CDN, defina `MOPGLED_USE_CDN=1`.

## Notificacoes
- Micro-servico em `_services/pushover/`.
- Pausar/ativar: `python3 _services/pushover/pushover.py disable` / `python3 _services/pushover/pushover.py enable`.

- `_docs/Prompts/`
- `_docs/playbooks/`
